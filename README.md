# power_bi-project
dashboard
1. Project Title & Overview
   Car_sales,
    This project is a Car Sales Dashboard that analyzes customer demographics, vehicle preferences, and dealer performance.
2. Objective
  The objective of this project is to evaluate sales performance by comparing MTD, YTD, and PYTD metrics. The dashboard highlights key indicators including total revenue, average vehicle price, sales growth, and number of units sold, enabling insights into trends across regions, dealers, and vehicle categories.
3. Dataset Description
   Source--kaggle
   Number of records-- approx 24kraw and 16 col
   Key columns:
   Customer Name, Gender, Annual Income
    Company, Model, Engine, Transmission
     Price ($), Dealer Name, Dealer Region,etc
4. Tools & Technologies
   Excel / Power BI/ Advance Dax
